FruchtsaftBot

Einfacher Twitch Chat Bot.

Der FruchtsaftBot ist 2015 als Spaßprojekt entstanden um die Zitate, die auf Fruchtsaft benannt sind in den Twitch Chat von Emerl (NoCrapTV) zu bringen.

Seitdem ist das Projekt stetig gewachsen und erledigt mittlerweile auch einfache Moderationsaufgaben, wie etwa das ausfiltern von Links.

Befehle

Befehle werden ganzh einfach mit vorangestellten ! in den Chat eines Channels geschrieben, indem sich der FruchtsaftBot befindet.

  • !anni
    codename

    Wird verwendet um ein beliebiges Zitat von oder über Anni aus zu geben.

    Codename

    Der codename Parameter akzeptiert jeden validen Codenamen für die Anni-Zitate.
    Wird ein falscher Codename angegeben, so wird ein zufälliges Zitat ausgegeben.

    Codenamen
  • !commands

    Aliase: !coms !help

    Zeigt den Link zu dieser Seite an.

  • !customCom
    action
    command
    response
    cooldown
    whitelist

    Fügt einen neuen benutzerdefinierten Befehl zu dem Channel hinzu, indem !customCom auferufen wird.

    Action

    Der action Parameter gibt die Aktion an, die FruchtsaftBot durchführen soll.
    Mögliche Werte sind:

    Aktion Verwendung
    list Gibt alle bereits definierten benutzerdefinierten des Channels an.
    Kann alternativ auch genauere Informationen zu einem einzelnen Befehl anzeigen.
    add Wird verwendet um einen neuen Befehl hinzuzufügen. whitelist
    edit Wird verwendet um einen bereits bestehenden Brfehl zu bearbeiten. whitelist
    remove Wird verwendet um einen neuen Befehl zu entfernen. whitelist
    Command

    command stellt den eigentlichen Befehl dar.

    Response

    response response wird lediglich mit den Aktionen add und edit verwendet um die Antwort des FruchtsaftBot anzugeben.
    Diese wird in doppelte Anführungszeichen "response" geschrieben.

    response akzeptiert zudem die Variablen {user}, welche den Displaynamen des Nutzers der den Befehl ausgeführt hat enthält, und {random}, welche einen zufälligen Nutzer im Chat benennt.

    Cooldown

    cooldown gibt die Zeitspanne in Sekunden an, die ein normaler Nutzer warten muss, um einen Befehl zweimal nacheinander ausführen zu können.
    Valide Werte sind natürliche Zahlen, oder das Flag -w, welches den Befehl nur für Nutzer auf der Whitelist ausführbar macht.

  • !infoText
    action
    text
    whitelist

    Wird verwendet, um die automatisch von FruchtsaftBot geposteten Informationen anzupassen.

    Action

    Der action Parameter gibt die Aktion an, die FruchtsaftBot durchführen soll.
    Mögliche Werte sind:

    Aktion Verwendung
    list Gibt alle für diesen Channel eingestellten Info-Texte an.
    FruchtsaftBot antwortet aus Gründen der übersicht in einem Whisper.
    add Wird verwendet um einen neuen Text hinzuzufügen.
    remove Wird verwendet um einen neuen Text zu entfernen.
    text

    text enthält für die Aktion add den Text, den FruchtsaftBot ausgeben soll.
    Für die Aktion remove muss der Zahlencode des zu entfernenden Textes angegeben weden. Dieser kann mit !infoText list erfragt werden.

  • !permit
    user
    whitelist

    Wird verwendet, um es einem Nutzer, der nicht auf der Whitelist steht zu erlauben einen beliebigen Link zu posten.

    User

    der user Parameter enthalt dern username des zu permittenden Nutzers.

  • !url

    Der Link zu dieser Seite.

  • !whitelist
    action
    user
    channel flag
    channel
    whitelist

    Wird verwendet um die Whitelist zu bearbeiten.

    Action

    Der action Parameter gibt die Aktion an, die FruchtsaftBot durchführen soll.
    Mögliche Werte sind:

    Aktion Verwendung
    list Gibt alle Nutzer an, die auf der Whitelist stehen.
    add Fügt der Whitelist einen neuen Nutzer hinzu.
    remove Löscht einen Nutzer von der Whitelist.
    User

    user ist der Nutzername, der geändert werden soll.

    Channel Flag

    Das channel flag wird verwendet, um eine andere Whitelist, asl die des aktuellen Channels anzupassen. Dabei wird unterschieden zwischen -c für einen anderen Channel und -g für die globale Whitelist.

    Channel

    Wenn channel flag auf -c gesetzt ist, kann mit channel die whitelist eines anderen Channels bearbeitet werden.
